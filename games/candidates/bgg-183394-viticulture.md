# Viticulture (2013)

## Solo mode
Viticulture solo uses the Automa system — a deck of cards drives an automated opponent that places workers on spaces according to priority rules. The automa blocks actions and scores points independently. Best played with Tuscany expansion for the full automa experience.

## What BGT would implement
- Tab Solo: automa card draw guide + worker placement priority reference
- Tab Scoring: player score + automa score comparison
- Tab Rules: automa rules reference
- Tab Setup: board preparation

## Feasibility
**Medium.** Automa priority rules are fixed — implementable as a reference + tracker.
