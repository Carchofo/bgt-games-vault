# Spirit Island (2017)

## Solo mode
Spirit Island has a native solo mode controlling 1 spirit. Highly complex: power cards, invader decks, island boards with 3 zones, blight track, fear track. Each round has 5 phases. An app tracking invader actions and fear would help significantly.

## What BGT would implement
- Tab Solo: fear tracker + blight counter + round tracker
- Tab Rules: invader phase reference (Ravage → Build → Explore)
- Tab Setup: island board preparation guide

## Feasibility
**Low.** Full implementation impractical — too many interacting systems. Partial tracker (fear/blight/round) is feasible as v1.

## Priority: #3 — Huge demand but HIGH complexity. Do simplified tracker only.
